# Research Notes

This directory contains reverse-engineering notes, format investigations, and implementation status documents that support `gputrace`.

These files are useful when extending parsers or validating Xcode parity, but they are not the primary user documentation for the CLI.

Start with:

- `TRACE_FORMAT.md` for capture bundle structure
- `RECORD_FORMATS.md` for MTSP record notes
- `BINARY_FORMAT_REFERENCE.md` for counter binary format notes
- `FIELD_OFFSET_QUICK_REFERENCE.md` for field lookup shortcuts
- `PERF_VS_NONPERF_TRACES.md` for capture mode differences
- `PERFCOUNTERS_STATUS.md` for counter support status
- `GPU_PROFILING_APIS_DISCOVERED.md` for profiler API notes
- `matching-xcode-gputools-parity.md` for feature parity tracking
